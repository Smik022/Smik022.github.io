ZeroTouch Onboarding

Product Manager•Femia Accountants

Femia Accountants ↗

Client onboarding for an Australian accounting firm, built against a hard AML/CTF compliance deadline.

Situation

Femia is a $5M accounting practice. New AML/CTF rules meant identifying, verifying and proving every client — all of it by hand.

Task

  • Replace three Excel calculators and paper forms with one system.
  • Keep pricing private from clients; write every check back to Karbon.

Result

Now runs the firm's client onboarding end to end — quote → intake → AML record → write-back into Karbon.

My Contribution

  • Split the flow in two: staff see pricing, clients never do. That constraint drove the architecture.
  • Ruled dry-run and live share one code path — Karbon writes are permanent, with no delete.
  • Caught that Karbon's number-typed Director ID field silently drops the leading zero from IDs starting 036.
